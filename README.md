# TaskManagement

簡單的任務管理返利

## 功能
- [x] 新增任務
- [x] 查詢任務
- [x] 刪除任務
- [x] 編輯任務

## 初階處理流程
- 基本的 MVC 流程開發
- 文件夾結構
- 表單處理
    - 建立表單 `Models`
    - 建立表單 `Views`
    - 建立對應 `Controller`
- 查詢處理
- 資料傳送處理
    - `Controller` 取值
    - 返回數值 `ViewModels`
- 快去暫存當作資料庫 `cache`
